7. Подача заявки на сервис

Заполните данные для получения виртуального номера

- Имя клиента должно совпадать с английским именем в паспорте (изменить нельзя).

- В качестве типа удостоверения личности выберите "Паспорт (护照)".

- При вводе контактной информации выберите страну "Other (其他)" и введите свой текущий номер телефона без кода страны.

- Если у вас есть реферал, введите его номер телефона. (Если нет — ввод не обязателен.)

- Только один номер телефона на один аккаунт WeChat (по одному для Китая и Гонконга).

- Введите официальный промокод, чтобы получить различные скидки, включая 7 дней бесплатного использования. [ Проверить промокод ]
