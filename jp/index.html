サービス利用方法

イーセンダー(eSender) 中国/香港バーチャル電話番号サービス

  • WeChatを起動
  • 申請情報を入力
  • 利用者情報に同意
  • バーチャル電話番号を発行
  • サービスを利用

WeChatをダウンロードして起動します。

1. WeChatモバイルアプリ

2. サービスページ検索

WeChat内の「通信録（通讯录）」メニューをクリックします。

2. サービスページ検索

WeChat内の「通信録（通讯录）」メニューをクリックします。

3. "eSender" サービス検索

上部の検索バーに「eSender」または「易博通」を入力して検索します。

検索結果から「易博通(eSender)」WeChat公式アカウント（微信公众号）を選択します。

3. "eSender" サービス検索

検索結果から「易博通(eSender)」WeChat公式アカウント（微信公众号）を選択します。

4. サービス登録と開通

WeChat公式アカウント（微信公众号）の下部メニューから「易博通 - 登记开通服务」をクリックします。

4. サービス登録と開通

WeChat公式アカウント（微信公众号）の下部メニューから「易博通 - 登记开通服务」をクリックします。

5. 希望する国を選択

「中国番号開通（开通中国内地号码）」または「香港番号開通（开通香港号码）」を選択します。

7. サービス申請

バーチャル電話番号発行に必要な利用者情報を登録します。

- 顧客名はパスポートに記載されている英語名と一致する必要があります。（修正不可）

- 身分証明書の種類は「パスポート（護照）」を選択します。

- 連絡先を入力する際、国は「その他（其他）」を選択し、電話番号は国番号なしで現在の自分の電話番号を入力します。

- 推薦者がいる場合は、推薦者の電話番号を入力します。（推薦者がいない場合は入力不要です。）

- 1つのWeChatアカウントにつき1つの電話番号のみ申請できます。（中国と香港それぞれ1つ）

- 公式プロモーションコードを入力すると、7日間の無料利用など、さまざまな割引特典を受けられます。[ プロモーションコード確認 ]

- 推薦者がいる場合は、推薦者の電話番号を入力します。（推薦者がいない場合は入力不要です。）
