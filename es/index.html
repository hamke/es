7. Solicita el servicio

Registra la información del usuario necesaria para emitir el número de teléfono virtual

- El nombre del cliente debe coincidir con el nombre en inglés del pasaporte (no se puede cambiar).

- Selecciona "Passport (护照)" como tipo de identificación.

- Al ingresar la información de contacto, selecciona "Other (其他)" como país e introduce tu número de teléfono actual sin el código de país.

- Si tienes un referente, introduce su número de teléfono. (Si no tienes referente, no es necesario ingresarlo.)

- Solo se puede solicitar un número de teléfono por cuenta de WeChat (uno para China y uno para Hong Kong).

- Ingresa el Código promocional oficial para recibir varios descuentos, como 7 días de uso gratis. [ Ver Código promocional ]
