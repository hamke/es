7. Solicite o serviço

Cadastre as informações do usuário necessárias para emitir o número de telefone virtual

- O nome do cliente deve coincidir com o nome em inglês do passaporte (não pode ser alterado).

- Selecione "Passport (护照)" como tipo de documento de identificação.

- Ao inserir as informações de contato, selecione "Other (其他)" como país e informe seu número de telefone atual sem o código do país.

- Se você tiver um referente, informe o número de telefone dele. (Se não tiver referente, não é necessário informar.)

- Só é possível solicitar um número de telefone por conta do WeChat (um para a China e um para Hong Kong).

- Insira o Código promocional oficial para receber vários descontos, como 15 dias adicionais de uso. [ Ver Código promocional ]
