7. Demander le service

Enregistrez les informations utilisateur nécessaires pour émettre le numéro de téléphone virtuel.

- Le nom du client doit correspondre au nom en anglais figurant sur le passeport (il ne peut pas être modifié).

- Sélectionnez "Passport (护照)" comme type de pièce d’identité.

- Lors de la saisie des informations de contact, sélectionnez "Other (其他)" comme pays et saisissez votre numéro de téléphone actuel sans l’indicatif du pays.

- Si vous avez un parrain (référent), saisissez son numéro de téléphone. (Si vous n’en avez pas, ce n’est pas nécessaire.)

- Un seul numéro de téléphone peut être demandé par compte WeChat (un pour la Chine et un pour Hong Kong).

- Saisissez le code promotionnel officiel pour bénéficier de diverses réductions, telles que 7 jours d’utilisation gratuite. [ Voir le code promotionnel ]
